# Weather Check Website

A simple Flask web application to check weather for any city, with support for Chinese and English interfaces.

## Features
- Input a city name to get current weather information
- Displays temperature, humidity, wind status, and rain status
- Bilingual interface (Chinese and English)

## How to Run
1. Ensure Python and Conda are installed.
2. Install dependencies: `conda install flask requests`
3. Run the app: `python app.py`
4. Open your browser and go to `http://127.0.0.1:5000`

## Usage
- Select language by clicking the link.
- Enter a city name (e.g., "New York", "北京").
- Submit to view weather details.
